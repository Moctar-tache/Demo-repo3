Today is a new day. Let's talk about how to edit a GitHub repo on your local machine.
To be able to do that, we can follow the steps below:
  1. Open your IDE first.
  2. Go to the directory where you want to load your repo.
  3. Go to your GitHub account and copy the HTTPS link of your repo. To get that link, click on "Code."
  4. Come back to your IDE and type the following command: `git clone "link you have copied"` and press Enter.
  5. After the previous step, you will have all your repo files on your local computer.

If you want to modify a file in your repo, follow the steps below:

  1. Open the file you want to edit, make your changes, and save it.
  2. Go to the terminal inside your IDE and write `git status` to see which files Git is tracking.
  3. Enter the command `git commit` followed by `-m "your comment or description"` and `-m "another comment"` and press Enter.
  4. Type `git add .` for all modified files or `git add filename` for a specific file.
  5. Type `git status` to confirm that the changes have been successfully made. The modified files will appear in green.
  6. To save your changes to the remote repository, type `git push origin "name of your branch"`.
